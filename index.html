Internet và World Wide Web

1. Internet là gì?

Internet là hệ thống mạng máy tính kết nối toàn cầu, cho phép hàng tỉ thiết bị trao đổi dữ liệu với nhau thông qua giao thức TCP/IP. Internet là hạ tầng vật lý và kỹ thuật bao gồm cáp quang, vệ tinh, bộ định tuyến (router)... phục vụ nhiều dịch vụ như email, truyền file (FTP), gọi điện video và cả World Wide Web.

2. World Wide Web (Web) là gì?

World Wide Web (Web) là một dịch vụ chạy trên Internet, cho phép người dùng truy cập và chia sẻ thông tin qua các trang web được liên kết bằng siêu liên kết (hyperlink). Web sử dụng giao thức HTTP/HTTPS và ngôn ngữ HTML để hiển thị nội dung trên trình duyệt. Web do Tim Berners-Lee phát minh năm 1991.

Nói cách khác: mọi trang Web đều dùng Internet, nhưng Internet không chỉ có Web.

3. Các khái niệm cơ bản

  • HTML – Ngôn ngữ đánh dấu siêu văn bản, dùng để tạo cấu trúc trang web.
  • HTTP / HTTPS – Giao thức truyền siêu văn bản giữa trình duyệt và máy chủ web.
  • URL – Địa chỉ duy nhất xác định một tài nguyên trên Web.
  • Trình duyệt (Browser) – Phần mềm dùng để truy cập và hiển thị trang web (Chrome, Firefox, Edge...).
  • Máy chủ Web (Web Server) – Máy tính lưu trữ và phục vụ các trang web cho người dùng.
  • DNS – Hệ thống phân giải tên miền, chuyển đổi tên miền thành địa chỉ IP.
