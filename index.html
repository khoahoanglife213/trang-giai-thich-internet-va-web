Internet và World Wide Web

Internet là hệ thống mạng máy tính kết nối toàn cầu, cho phép các thiết bị trao đổi dữ liệu với nhau qua giao thức TCP/IP.

World Wide Web (Web) là dịch vụ chạy trên Internet, dùng để truy cập các trang web qua trình duyệt bằng giao thức HTTP. Web chỉ là một phần của Internet.

  • HTML - ngôn ngữ tạo cấu trúc trang web
  • HTTP - giao thức truyền dữ liệu web
  • URL - địa chỉ định vị tài nguyên trên web
